## Khởi chạy dự án

Chạy theo các bước:

#### `git clone https://github.com/hungds99/SocialApp-Clone.git`
#### `cd SocialApp-Clone`
#### `npm install`
#### `npm start`

Chạy ứng dụng ở chế độ phát triển.\
Mở [http://localhost:3000](http://localhost:3000) để xem nó trên trình duyệt.
